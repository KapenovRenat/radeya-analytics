# РНП — Рука на пульсе

> Система еженедельного контроля рекламных кампаний Kaspi.kz. Excel-файл с понедельной динамикой по всем кампаниям и товарам.

## Содержание

### Рабочий файл

`РНП_Каспи_магазина_с_кампаниями_3.xlsx` — 5 листов:

| Лист | Период | Назначение |
|---|---|---|
| Сводка | 18.04–17.05.2026 | Итоги месяца: план/факт, топ/проблемные |
| Сводка Кратка | 27.04–18.05.2026 | Понедельная динамика за 3 недели |
| Рекламные компании Май | 27.04–24.05.2026 | **Основной** — ~48 кампаний, 4 недели |
| Рекламные компании | архив | Старый лист |
| По товарам | 18.05–24.05.2026 | Товары внутри кампаний по категориям |

### Структура листа «Рекламные компании»

**Фиксированные колонки (A–G):**
- Название кампании
- Статус рекламы (Включена / Выключена)
- Улучшить карточку? / Есть отзывы? / Есть скидка? / В наличии? / Есть видео?

**Колонки на каждую неделю (повторяются блоком):**
- Дата начала / Дата выключения
- Потрачено (тг) — из CSV
- Дн. бюджет (тг) — вручную
- Ср. клик (тг) — из CSV
- Заказы / ДРР%
- CTR%
- Конв. клик → корзину%
- Конв. клик → избранное%
- Оценка (🟢 Хорошо / 🟡 Нормально / 🔴 Плохо)

**Итого за месяц:** Потрачено, Выручка, Заказов, ДРР%, CTR%, Конв→корзину%, Конв→избр.%, Оценка

### Структура листа «По товарам»

Товар → Категория → Статус → флаги → недельные метрики.
Группировка по категориям (Диваны, Кресла, и т.д.).
649 строк = каждый SKU (цвет/размер) в каждой кампании.

### Цикл работы (каждый понедельник)

1. Скачать CSV из Kaspi кабинета (по кампаниям + по товарам)
2. Вставить данные в новую неделю в Excel
3. Проверить оценки — проставляются по формулам
4. Прочитать Сводку — топ и проблемные
5. Принять решения по бюджетам

### Кабинеты

Активны **два кабинета**:
1. **Основной** — все ~48 кампаний по товарам
2. **Внешняя реклама** — подключён с 18.05.2026
