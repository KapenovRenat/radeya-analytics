# Фича: Раздел РНП в приложении

> Планируемый раздел `/stores/[id]/rnp` для еженедельного контроля рекламных кампаний прямо в приложении вместо Excel.

## Содержание

### Цель

Заменить ручной Excel-файл РНП встроенным разделом приложения.  
Загрузка CSV из Kaspi → автоматические метрики и оценки → недельная динамика.

### Статус

**Запланировано** — план согласован 2026-05-28, реализация не начата.

### БД — новые таблицы

| Таблица | Назначение |
|---|---|
| `rnp_campaigns` | Справочник кампаний (название, статус, флаги) |
| `rnp_weekly_stats` | Метрики кампании по неделям |
| `rnp_product_stats` | Метрики по товарам внутри кампании |

**`rnp_campaigns`:**
`id, store_id, name, status, improve_card, has_reviews, has_discount, in_stock, has_video`

**`rnp_weekly_stats`:**
`id, campaign_id, week_start, week_end, spent, daily_budget, avg_click, orders, drr_pct, ctr_pct, conv_to_cart_pct, conv_to_fav_pct, rating`

**`rnp_product_stats`:**
`id, campaign_id, week_start, product_name, category, status, spent, avg_click, orders, drr_pct, ctr_pct, rating`

### Загрузка данных

- Upload CSV из Kaspi кабинета (drag & drop)
- Парсинг двух форматов: «По кампаниям» и «По товарам»
- Upsert — повторная загрузка не дублирует
- Структура CSV → [[kaspi-rnp]]

### UI — 3 вкладки

**Сводка `/stores/[id]/rnp`**
- KPI: суммарный расход, средний ДРР%, заказов из рекламы, CTR%
- Таблица кампаний: последняя неделя vs предыдущая (стрелки ↑↓)
- Цветовые оценки: 🟢🟡🔴

**По кампаниям `/stores/[id]/rnp/campaigns`**
- Список кампаний с раскрытием
- График недельной динамики: расход, ДРР%, заказы
- Сравнение недель в таблице

**По товарам `/stores/[id]/rnp/products`**
- Выбор кампании → список SKU с метриками
- Группировка по категориям (Диваны, Кресла и т.д.)
- Фильтр по оценке (только 🔴 Плохо)

### Рабочий процесс пользователя

1. Каждый понедельник скачать CSV из Kaspi кабинета
2. Загрузить в приложение через Upload
3. Смотреть аналитику и принимать решения по бюджетам
